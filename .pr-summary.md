## Qué hace este PR (en lenguaje no técnico)

Este PR añade el gate G13 «Scope-trace audit» al pipeline de pre-vuelo de pr-plan. El gate verifica automáticamente que cada archivo modificado en un PR tiene una razón de ser en los criterios de aceptación del spec que justifica ese PR — evitando que cambios colaterales («scope-creep») se cuelen silenciosamente en el código sin que nadie los detecte hasta la revisión humana. El mecanismo es completamente determinístico: sin llamadas a ningún modelo de IA, sin dependencias externas, menos de 1 segundo de overhead.

Spec ref: SE-079
