# PR Summary — SE-150 Pre-output validator (TTSR-inspired)

This PR adds a static pre-output validator inspired by TTSR patterns. It covers 7 security and safety rules (POR-001: hardcoded PAT, POR-002: CLAUDE_PROJECT_DIR, POR-003: git commit on main, POR-004: terraform apply without guard, POR-005: rm -rf root, POR-006: inline password, POR-007: git push --force outside agent/* branches). The validator script supports check, list-rules, and explain subcommands. It fills gaps not covered by existing bash-only hooks: Write and Edit tool content is validated before output. Zero exact duplicates with existing hooks; 6 of 7 rules are complementary. Includes 34 BATS tests (34/34 passing).
