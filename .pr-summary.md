# PR Summary — SE-148 AgentRunSummary telemetry

This PR introduces structured telemetry for agent runs. It adds a schema v2 JSONL format with 13 fields (run_id, slug, ts_start, ts_end, duration_s, tools_invoked, tools_unused, tool_status, models_used, tokens_in, tokens_out, cost_usd, run_status), two scripts (agent-run-logger.sh for recording runs and agent-run-report.sh for human-readable summaries), and 15 BATS tests covering the full schema, subcommands, and backward compatibility with v1 records. The schema is designed to be backward-compatible so existing telemetry data continues to work.
