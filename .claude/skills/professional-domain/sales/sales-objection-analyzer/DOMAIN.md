# Sales Objection Analyzer — Dominio y Conocimiento

## Por qué existe este skill

El 80% de las objeciones no son lo que parecen. "Es muy caro" suele significar
"no veo el valor suficiente". "No es el momento" suele significar "no es prioridad
para quien manda". Un comercial que responde a la objeción literal en lugar de
a la causa raíz pierde el deal o lo retrasa innecesariamente.

---

## Taxonomía de objeciones comerciales

### PRECIO — Objeciones relacionadas con el coste

**Formas típicas**:
- "Es demasiado caro para nuestro presupuesto"
- "Tenemos opciones más baratas"
- "No puedo justificar esta inversión"

**Causa raíz más frecuente**: el valor percibido no justifica el precio.
El problema no es el precio — es que el cliente no ha entendido (o no le hemos
explicado) cuánto le cuesta NO resolver el problema.

**Respuesta tipo**: cuantificar el coste del problema antes de hablar de precio.
Si el problema cuesta 200k€/año, 50k€ de inversión no es caro — es un 4x ROI.

**Pregunta de seguimiento**: "¿Qué tendría que ser cierto para que esta inversión
tuviera sentido para vosotros?"

---

### TIMING — Objeciones relacionadas con el momento

**Formas típicas**:
- "No es el momento, hay mucho lío ahora"
- "Esperemos al Q1/Q3/año que viene"
- "Estamos en medio de otro proyecto"
- "Tenemos que terminar primero con X"

**Causa raíz más frecuente**: no es prioritario para quien decide. El "timing"
es una objeción de bajo riesgo para el cliente — no dice que no, no quema el puente.

**Regla de oro**: Si hay un Champion interno real, el timing se negocia.
Si no hay Champion, el timing es el pretexto para no decidir.

**Pregunta de seguimiento**: "¿Qué tiene que pasar para que esto sea prioritario?
¿Hay una fecha límite asociada al problema que estáis teniendo?"

---

### COMPETENCIA — Objeciones relacionadas con alternativas

**Formas típicas**:
- "Estamos mirando otras opciones también"
- "Tenemos una propuesta de X que es más completa"
- "Ya tenemos una solución para eso"
- "Lo hacemos internamente"

**Causa raíz más frecuente**: no tenemos diferenciación clara o no hemos llegado
al Economic Buyer (el Champion nos está protegiendo de la decisión real).

**Respuesta tipo**: no atacar a la competencia. Preguntar qué criterios de
decisión son más importantes y reposicionar en torno a los que se ganan.

**Señal de alerta**: si la alternativa es "seguir como estamos" o "hacerlo
internamente", el deal puede estar perdido si no hay un Event Trigger claro.

---

### RIESGO — Objeciones relacionadas con incertidumbre o miedo al cambio

**Formas típicas**:
- "No estamos seguros de que funcione para nosotros"
- "Hemos tenido malas experiencias con implementaciones anteriores"
- "¿Qué pasa si no funciona?"
- "Somos un caso especial, no sé si aplica"

**Causa raíz más frecuente**: falta de confianza (en la solución, en el proveedor,
en la capacidad interna para gestionar el cambio).

**Respuesta tipo**: referencias de clientes similares + plan de mitigación de riesgo
concreto + garantías de servicio. Nunca minimizar la objeción ("eso es muy normal").

**Pregunta de seguimiento**: "¿Qué necesitaríais ver para sentiros seguros con
esta decisión?"

---

### INTERNO — Objeciones relacionadas con proceso o stakeholders internos

**Formas típicas**:
- "Tengo que consultarlo con mi jefe / con IT / con el comité"
- "Necesito aprobación de compras"
- "No puedo decidir yo solo"
- "Tenemos un proceso de homologación de proveedores"

**Causa raíz más frecuente**: no hemos llegado al Economic Buyer. El interlocutor
actual no tiene autoridad de decisión. O: el interlocutor está de acuerdo pero
no tiene el mandato para ir al siguiente paso sin cobertura.

**Regla crítica**: una objeción INTERNO en etapa de Propuesta o Negociación indica
que el proceso MEDDIC está incompleto — no sabemos quién es el Economic Buyer real.

**Respuesta tipo**: solicitar una reunión conjunta con el decisor real, ofreciendo
apoyo para la presentación interna.

---

### OTRO — Objeciones no clasificables

Captura residual para objeciones que no encajan en las anteriores.
Puede incluir: problemas de integración técnica específica, restricciones
regulatorias, limitaciones contractuales previas.

---

## Regla de oro de las objeciones

> El 80% de las objeciones indican discovery incompleto, no cierre fallido.

Si aparecen muchas objeciones en etapa de Propuesta o Negociación, el problema
no es la propuesta — es que el discovery (calificación) fue insuficiente.

Las objeciones más peligrosas son las que no se expresan: el cliente que no
tiene objeciones aparentes pero tampoco avanza.

---

## Señales de que el deal está perdido (no recuperable)

| Señal | Descripción |
|---|---|
| Economic Buyer no accesible | Nunca se puede hablar con quien decide |
| Silencio prolongado | No responden emails ni llamadas tras propuesta |
| "Seguiremos con lo actual" sin Event Trigger | No hay razón para cambiar ahora |
| Objeción PRECIO tras propuesta ajustada | El precio no era el problema real |
| Competidor seleccionado informalmente | "Estamos muy avanzados con X" |
| No hay Champion interno | Nadie nos defiende en el proceso interno |

Cuando aparecen 3 o más señales, la recomendación es preguntar directamente:
"¿Tiene sentido que sigamos avanzando? Prefiero saberlo ahora para no hacer
perder el tiempo a nadie."
