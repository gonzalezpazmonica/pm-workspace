# Plantilla Visual — Esquema de Colores Corporativo

## Paleta Principal

| Elemento | Color | Hex |
|----------|-------|-----|
| Fondo corporativo | Azul | #0078D4 |
| Fondo secundario | Gris claro | #F3F3F3 |
| Semáforo Verde | Verde | #00B050 |
| Semáforo Amarillo | Amarillo | #FFC000 |
| Semáforo Rojo | Rojo | #FF0000 |
| Cabecera tablas | Azul oscuro | #003865 |
| Texto principal | Negro | #000000 |
| Texto secundario | Gris | #666666 |

## Aplicación en Diapositivas

- **Portada:** Fondo azul corporativo, título blanco, logo esquina superior derecha
- **Diapositivas de contenido:** Fondo blanco, barra superior azul con título blanco
- **Numeración:** Inferior derecha, formato: "Página X de Y"
- **Gráficos:** Paleta corporativa, fuente Calibri 10pt, leyenda a la derecha
- **Tablas:** Cabecera azul oscuro (#003865) con texto blanco, filas alternas blanco/#F3F3F3
- **Semáforos:** Círculos grandes (40px) + texto de estado debajo

## Fuentes

- Título principal: Calibri Bold 44pt
- Títulos sección: Calibri Bold 28pt
- Contenido: Calibri Regular 12pt
- Pie de página: Calibri Regular 10pt
