### Added
- **G13 Scope-trace audit** (`scripts/pr-plan-gates.sh`) — SE-079.
  Pre-push gate that verifies every changed file traces back to an acceptance
  criterion of the spec referenced in the PR. Emits FAIL with `file → NO MATCH`
  table when scope-creep is detected; WARN-only when no spec is linked. Supports
  multi-spec PRs, hard-coded whitelist (CHANGELOG.d/, .scm/, .confidentiality-
  signature, .pr-summary.md), and `Scope-trace: skip — <reason>` override.
  Pure-bash, zero LLM calls, <1s overhead. 25 BATS tests (100% pass).
  Doc: `docs/rules/domain/pr-plan-scope-trace.md`.
