# Narrative Structure — Pyramid principle para PPT directiva

## Regla maestra

**Una idea por slide. Idea principal arriba (título), soporte abajo (cuerpo).**

Si necesitas 2 ideas, son 2 slides. Si necesitas 0.5 ideas, fusiona con la siguiente.

## Estructura canónica de 11 slides

| # | Función | Patrón visual |
|---:|---|---|
| 1 | Portada | Imagen + texto |
| 2 | Problema / Por qué ahora | 3-4 cajas, datos |
| 3 | Tesis / "Qué hace X" | Texto principal + visual |
| 4 | Pilar 1 (capacidad principal) | 7 cajas o pipeline |
| 5 | Pilar 2 | 2 columnas o 3 cajas |
| 6 | Pilar 3 | Pipeline o flujo |
| 7 | Resumen pilares | 3 cajas |
| 8 | Prueba / Caso real | 4 cajas 2×2 |
| 9 | Cómo encaja con su mundo | 4 cajas + callout |
| 10 | Filosofía / Principios | 3 cajas |
| 11 | CTA / Cierre | Imagen + texto + firma |

## Titulares: reglas

| MAL | BIEN |
|---|---|
| "Memoria" | "Savia separa lo que sabe de lo que decide." |
| "Soberanía de datos" | "Los datos de acme-corp no salen de acme-corp." |
| "Code Review Court" | "Ninguna decisión de un agente llega sin pasar por un tribunal." |
| "Conclusión" | "Savia, preparada para acompañarte." |

Reglas:
- Verbo en presente afirmativo.
- Sujeto explícito.
- Concluye con punto.
- Máx 2 líneas a 28pt.

## Cuerpo: reglas

- 3-5 bullets máximo por caja.
- Cada bullet ≤12 palabras.
- Sin jerga interna ("hooks", "PreToolUse", "L0-L3") salvo si es lo que se quiere explicar.
- Datos concretos > abstracciones ("4 capas" > "varias capas").

## Frases prohibidas

- "Revolucionario" / "disruptivo" / "best-in-class"
- "Solución 360" / "end-to-end" / "holístico"
- "Empoderar" / "transformar digitalmente"
- "Líder del mercado" (a menos que sea cuantificable y verificable)

## Datos del cliente

- Si se incluyen datos sin anonimizar → confirmación explícita PM.
- Si son N4b (PM-only) → NUNCA en slide cliente.
- Conteos internos del workspace (commands, agents, skills) → NUNCA como KPIs cliente.

## Cierre: firma del producto

Si la herramienta tiene "voz" (caso Savia):
- Tono profesional + humano, sin diminutivos cariñosos delante de directivos.
- Texto corto (4-6 líneas).
- Cierra con frase fuerza ("Vuestro proyecto, vuestras reglas.").
- Firma "— [Nombre producto]" en color acento.
